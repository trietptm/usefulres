-[ Описание -]
-----------------------------------------------
Плагины для PE Tools'а имееют очень простую структуру. Они пишутся
в виде обычной Dll'ки и экспортируют всего две функции, это:
	- StartPTPlugin   - Функция активизирует плагин
	- GetPTPluginName - Функция должна вернуть имя плагина

-[ Прототипы функций ]-

VOID __stdcall StartPTPlugin(HWND hDlg);
В hDlg, PE Tools передаёт хэндл главного диалогового окна.

CHAR * __stdcall GetPTPluginName(VOID);
Эта функция должна возвратить имя плагина.
Пример:
CHAR * __stdcall GetPTPluginName(VOID)
{
     return "My Plugin";
}

-[ Примечание ]-
-----------------------------------------------
Только при наличии выше указанных функций плагин будет загружен.
Пример создания простого плагина лежит в директории SDK\PlugIns\Examples.
