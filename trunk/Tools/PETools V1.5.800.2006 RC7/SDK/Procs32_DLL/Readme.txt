 I. Описание:
--------------
	Procs32.dll - это Dll-ка для получения списка запущенных
процессов/модулей под Win9x/2k/NT/XP. Dll-ка использует API функции
двух модулей: Kernel32.dll (для Win9x) и PSAPI.DLL (для NT).


II. Примечание:
--------------
	Для работы Dll-ки под системами линейки NT, необходим файл
PSAPI.DLL.
