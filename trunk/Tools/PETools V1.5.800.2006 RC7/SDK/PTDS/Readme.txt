-[ Описание -]
-----------------------------------------------
PTDS - Это интерфейс для написания плагинов для PE Tools версий 1.5++
Он имеет следующие возможности:
	- Дапм процесса/модуля процесса
	- Получение списка процессов/модулей процесса
	- Оптимизация PE файлов (RebuildPE)

-[ Система интерфейса обмена информацией -]
-----------------------------------------------
Обмен инфой между плагином и PTDS осуществляется при помощи функции
'SendMessage'. Но сначала необходимо получить хендл окна PTDS. Для
этого используется функция 'FindWindow' с параметром PTDS_WND_NAME.

void SendMessage(
  HWND hWnd,        - Хэндл окна PTDS
  UINT Msg,         - WM_PTDS_CMD_XXX команда
  WPARAM wParam,    - PID плагина
  LPARAM lParam     - указатель на PTDS_XXX структуру
);

-----------------------------------------------
PTDS SDK пока написана только для MSVC/ASM/Delphi.
Позже будут и другие языки. Для более подробного изучения, используйте
примеры из SDK. Пожалуйста вышлите мне Ваши плагины (NEOx@Pisem.net).
