_______________________ eXtreme dumper _______________

eXtreme dumper - это новая примочка к PE Tools позволяющая легко дампить то, что раньше считалось нетривиальной задачей.
Например, можно легко сдампить Extreme Protector (Themida) всех версий, или смело забить на CopyMem при дампе армадилы.
Можно дампить даже память ядра системы (ядро, системные длл, драйвера).
Короче, штука это в хозяйстве полезная, и может кому-нибудь да пригодится.

======================================================
Методы дампа:

Dll Injection - применять против CopyMem в армадиле, в процесс внедряется dll, которая дампит его изнутри.
Для использования этой опции нужно подключить длл к процессу, это можно сделать выбрав процесс в списке и нажав Attach или выбрав и запустив файл через плагин (позволяет обойти перехват ZwCreateThread).

Ring0 - чтение памяти с помощью драйвера, позволяет обойти антидамповую защиту икспрота и фемиды.

======================================================
Опции дампа:

EnableOpenProcessEx - получение хэндлов без использования OpenProcess, позволяет обойти перехват ZwOpenProcess протектором.

Protect PE Tools - защитить процесс PE Tools от воздействия со стороны других процессов, это полезно тогда, когда защита просто прибивает запущенный PE Tools.

Enable kernel modules dumping - включить поддержку дампа модулей ядра, после этого вся ядерная память будет видна как память процесса system и можно будет дампить загруженные в нее драйвера.
